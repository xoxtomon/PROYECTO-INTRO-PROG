Este programa necesita varios requerimientos:
	1. necesita estar ubicado en la siguiente direccion: C:\temp\ARCHIVOS2
	2. En su defecto se deben modificar las rutas dentro de la aplicación para que pueda encontrar los archivos
Esto debido a que fue imposible para mi hacerlo 'Auto contenido', no pude encontrar solución en internet, y tampoco pude

Guia de uso:
la aplicación tiene varias pestañas, cuando una ruta necesite que haya un usuario registrado y no lo hay esta redireccionara para 
que el usuario inicie sesion, o haga registro en su defecto. A traves de tags 'select' el usuario pueee escoger sus opciones esto para
ahorrar la verificación que es más complicada, y tambien para que si eventualmente se actualizan los datos del archivo
'datos.txt' la aplicación los incluya de forma natural. Por otra parte la aplicación es muy sencilla y es intuituva debdo a la naturaleza de esta
Sin embargo, cumple con la mayoria de los requerimientos.
